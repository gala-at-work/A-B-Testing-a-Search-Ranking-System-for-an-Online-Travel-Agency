# A-B-Testing-a-Search-Ranking-System-for-an-Online-Travel-Agency
In this project, I take on the role of a data scientist at an online travel agency that has developed a new search ranking system intended to improve booking conversion. Rather than relying on intuition, I evaluate the system through a controlled A/B test, in which users are randomly divided into a control group exposed to the existing system and a treatment group exposed to the new one. Using Python, I examine the experiment data, verify that the test was set up fairly, measure the difference in conversion rates between the two groups, and apply statistical testing to determine whether any observed improvement reflects a genuine effect rather than random chance. I then translate these findings into a clear, evidence-based recommendation on whether the company should roll out the new system
